## 🧭 UX Research & Problem Statement
*   **The Problem:** Fitness apps lose user engagement because logging daily data manually feels like a boring chore.
*   **The Solution:** Implemented streak-based gamification rewards and a visual widget dashboard that updates progress in real-time with minimum typing.

---

## 🎨 Design System
*   **Typography:** Plus Jakarta Sans (Sporty, energetic, and clean)
*   **Color Palette:**
    *   Energy Neon: `#CCFF00` (Lime Green for active vibes)
    *   Background: `#121826` (Premium dark mode tech theme)
*   **Key Features:** Interactive progress charts, drag-and-drop workout planners, and animated habit-tracking checklists.
